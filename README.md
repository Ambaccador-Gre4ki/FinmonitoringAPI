# FinMonAPI
Программа для взаимодействия с API Росфинмониторинга, версия 0.1.0.

Позволяет автоматически скачивать список террористов (перечни ООН доступны в продуктивной среде)

## Что это такое?
Консольное приложение, которое авторизируется на портале Росфинмониторинга и скачивает оттуда два файла: справочник ООН и перечень террористов-экстремистов

Данная программа нужна в связи с изменениями в ФЗ №462 от 15.12.2025, согласно которому применять меры необходимо в течение 24 часов, а не в течение одного рабочего дня.

## Как это работает?
Вы передаёте в программу данные сертификата, с помощью которого авторизируетесь на сайте, а также логин и пароль. Далее она автоматически проверяет наличие перечней и скачивает их.
>:warning: **ВНИМАНИЕ:** на данный момент проверки скачивания одних и тех же справочников нет; просто загрузка с портала
