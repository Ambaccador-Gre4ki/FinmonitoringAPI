# FinMonAPI
Программа для взаимодействия с API Росфинмониторинга, версия 1.1.0

Позволяет автоматически скачивать список террористов и перечни ООН
## Что это такое?
Консольное приложение на .NET 9.0, которое авторизуется на портале Росфинмониторинга и скачивает оттуда справочники ООН и перечень террористов-экстремистов

Данная программа нужна в связи с изменениями в ФЗ №462 от 15.12.2025, согласно которому применять меры необходимо в течение 24 часов, а не в течение одного рабочего дня. Изменения вступают в силу с 14.06.2026

## Как это работает?
Вы передаёте в программу данные сертификата, с помощью которого авторизуетесь на сайте, а также логин и пароль. Далее она автоматически проверяет наличие файлов и скачивает их.
>:warning: **ВНИМАНИЕ:** на данный момент проверки скачивания одних и тех же справочников нет

## Системные требования
* ОС - Windows 10 (x64)
* Установленный СКЗИ КриптоПро CSP
* Наличие сертификата в личном хранилище текущего пользователя

## Инструкция по запуску
1. Скачайте архив и распакуйте его
2. Откройте файл `appsettings.json` и укажите в полях:
   * Thumbprint - отпечаток сертификата для доступа в личный кабинет
   * Login - логин для входа в личный кабинет
   * Password - пароль для входа в личный кабинет
   * DownloadFolder - путь до папки, куда будут сохранятся файлы
   * LogsFolder - путь до папки, куда будут записываться логи
3. Запустите `FinMonAPI.exe`. Результаты работы будут отображаться и в окне консоли, и в текстовом файле

## Разница между test и prod версиями
`test` предназначен для прохождения тестирования путём вызова тестовых методов. Также в этой версии реализовано получение "сырых" JSON-конвертов запросов и ответов, т.к. они тоже нужны для прохождения тестирования. **Скачивание справочников ООН** в этой версии **не предусмотрено**! 

`prod` используется для обычной работы. В этой версии уже есть возможность скачивать справочники ООН и убраны лишние отладочные методы и получение "сырых" JSON'ов
